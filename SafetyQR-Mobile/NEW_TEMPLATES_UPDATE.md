## 🛠️ Troubleshooting Tunnel
If `npx expo start --tunnel` fails:
1.  Try running just `npx expo start`.
2.  Use the "LAN" IP if your phone is on the same Wi-Fi.
3.  Restart your PC if ngrok is stuck.
